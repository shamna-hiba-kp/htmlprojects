PROGRAMMING LANGUAGES

Programming language is a formal language designed to communicate instructions to a machine, particularly a computer. Programming languages can be used to create programs that control the behaviour of a machine and/or to express algorithms precisely.

Java

Java is a computer programming language that is concurrent, class-based, object-oriented and specificlly designed to have as few implentation dependencies as possible. It is intended to let application developers "write once, run anywhere" (WORA), meaning that code runs on one platform does not need to be recompiled to run on anaother.
